# 阿里云OSS配置说明

## 重要提示
⚠️ 请勿在此文件中存储真实的 AccessKey!

## 配置步骤

### 方式一:使用环境变量(推荐)
1. 设置系统环境变量:
   - OSS_ACCESS_KEY_ID = 你的AccessKey ID
   - OSS_ACCESS_KEY_SECRET = 你的AccessKey Secret

2. 代码会自动从环境变量读取凭证

### 方式二:使用Spring Cloud Config或Nacos等配置中心

### 方式三:本地开发测试(仅用于开发环境)
在 application.yml 中配置(但务必将 application.yml 加入 .gitignore):
```yaml
aliyun:
  oss:
    endpoint: https://oss-cn-beijing.aliyuncs.com
    bucketName: your-bucket-name
    region: cn-beijing
```

## 安全最佳实践
- ✅ 使用环境变量或密钥管理服务
- ✅ 定期轮换 AccessKey
- ✅ 为不同环境使用不同的密钥
- ❌ 不要将密钥硬编码在代码中
- ❌ 不要将包含密钥的文件提交到 Git
- ❌ 不要在日志中打印密钥
